1) npm i create-react-app -g
это просто пакет для создания реакта
который поставился глобально на компьютер (в папку ноде модулес на диске С) 
Это команда которая устанавливает сборку фейсбука на компьютер, и теперь мы можем открывать 
готовые реакт проекты (дает возможность создать новую папку с новым проектом на реакте, 
со всеми нужными зависимостями, папками, пакеты)

если нпм версии 5.2 и выше нужно использовать команду
npx npm i create-react-app -g


2) create-react-app intro_react
создаем стандартную сборку реакта (create-react-app) и название папки придумываем (intro_react)

3) React Extension Pack
в VS code расширение устанавливаем, и перезагружаем реакт
